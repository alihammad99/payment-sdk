Payment SDK

A unified, type-safe Node.js SDK for integrating multiple payment providers. Simplify your payment processing with a consistent API across Alqasa, ZainCash, Stripe, and AsiaPay.

Installation

Install the package using npm:

npm install payment-sdk

Quick Start

Here's how to get started with the SDK:

import { AlqasaClient } from 'payment-sdk';

// Initialize the client
const client = new AlqasaClient('your-api-token');

// Create a payment
const payment = await client.createPayment({
    amount: 5000,
    currency: 'IQD',
    orderId: 'ORD-001'
});

Alqasa Integration

Alqasa is a primary provider for local credit card payments.

Configuration

// Option 1: Using Token
const client = new AlqasaClient('TOKEN');

// Option 2: Using Credentials
const client = new AlqasaClient({
    clientId: 'CLIENT_ID',
    clientSecret: 'SECRET'
});

Create Payment

const response = await client.createPayment({
    amount: 1000,
    currency: 'IQD',
    description: 'Test Order',
    orderId: '12345'
});

ZainCash Integration

Support for ZainCash e-wallet payments.

import { ZaincashClient } from 'payment-sdk';

const zain = new ZaincashClient({
    production: false,
    merchantId: 'MERCHANT_ID',
    secret: 'SECRET_KEY',
    token: 'JWT_TOKEN'
});

const tx = await zain.initTransaction({
    amount: 2500,
    serviceType: 'Goods',
    msisdn: '9647800000000',
    orderId: 'ORDER_002',
    redirectUrl: 'https://mysite.com/callback'
});

Stripe Integration

Stripe integration using Checkout Sessions for international card payments.

Configuration

import { StripeProvider } from 'payment-sdk';

const stripe = new StripeProvider();
await stripe.init({
    secretKey: 'sk_test_YOUR_SECRET_KEY'
});

Create Payment

const response = await stripe.createPayment({
    amount: 5000,
    currency: 'usd',
    description: 'Product Purchase',
    orderId: 'ORDER_789',
    callbackUrl: 'https://yoursite.com/success',
    customer: {
        email: 'customer@example.com'
    }
});

// Returns session with redirectUrl to Stripe Checkout
console.log(response.redirectUrl);

Check Payment Status

const status = await stripe.checkStatus(transactionId);
console.log(status.status); // 'COMPLETED', 'PENDING', or 'FAILED'

AsiaPay Integration

AsiaPay payment gateway for Asian market payment processing.

Configuration

import { AsiapayProvider } from 'payment-sdk';

const asiapay = new AsiapayProvider();
await asiapay.init({
    production: false,
    appId: 'YOUR_APP_ID',
    appSecret: 'YOUR_APP_SECRET',
    merchCode: 'YOUR_MERCHANT_CODE',
    privateKey: 'YOUR_PRIVATE_KEY'
});

Create Order

const response = await asiapay.createPayment({
    amount: 10000,
    currency: 'IQD',
    orderId: 'ORDER_999',
    description: 'Order Payment',
    callbackUrl: 'https://yoursite.com/callback',
    notifyUrl: 'https://yoursite.com/webhook'
});

console.log(response.redirectUrl);

Error Handling

The SDK throws structured errors that you should catch and handle.

try {
    await client.createPayment({...});
} catch (error) {
    if (error.message) {
        console.error('API Error:', error.message);
    } else {
        console.error('Unknown Error:', error);
    }
}
